# E-commerce (Amazon-style Clone)

A front-end e-commerce website project built with HTML, CSS, and JavaScript, designed to mimic the look and feel of an Amazon-style storefront.

## What this project includes
- A product listing page
- A shopping cart experience
- A checkout page

## Project structure
- `amazon.html` — main storefront page
- `checkout.html` — checkout flow page
- `styles/` — CSS files
- `scripts/` — JavaScript files
- `assets/` — images, icons, and ratings

## How to run
Open the project in a browser by launching `amazon.html` directly, or use a simple local server such as Live Server in VS Code.

## Status
This project is still a work in progress and may change as development continues.
